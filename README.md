## Setup Instructions
1. Clone repo
2. Install dependencies: `pip install -r requirements.txt`
3. Run migrations: `flask db upgrade`
4. Seed data: `python seed.py`
5. Run app: `python run.py`

## Running Demo
`python demo.py`

## Running Tests
`pytest tests/ -v`
